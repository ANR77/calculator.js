## Install

### :bird: Bower<br>![](https://badge.fury.io/bo/material_calculator.svg)

```
bower install material_calculator
```

### :package: NPM<br>![](https://badge.fury.io/js/material_calculator.svg)

```
npm i material_calculator
```
